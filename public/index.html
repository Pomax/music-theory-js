Waiting for Web MIDI to kick in...

Hook up a MIDI controller and see if the "led" turns green. If it does: you're good to go, go start playing and slide some draw bars! Tweak some knobs, see what happens! Note that this works in Chrome, but not "out of the box" in Firefox, mostly because Firefox has been spending the last six years writing comments instead of just implementing the core for Web MIDI so you need a plugin. I know, right? Crazy, but easy enought to add (note that it links to 2 more things you'll need installed. I thought that was worth it. Maybe you don't. Use Chrome if you think addons are too much work).
