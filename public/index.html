Waiting for Web MIDI to kick in...

Hook up a MIDI controller and see if the "led" turns green. If it does: you're good to go, go start playing and slide some sliders and tweak some knobs on your controller, see what happens! Don't see anything happen? Click on a slider's label to "CC learn" a different controller, just like you would in any regular old VST instrument, and see what happens, v2!

Note that this works out of the box in Chrome, but not in Firefox, mostly because Firefox is still in the process of figuring out their implementation approach for Web MIDI. It's been six years, but no one's paid them to implement it, so... you get what you pay for? Thankfully, the web midi extension for Firefox lets you add Web MIDI support pretty easily, although it does need 2 more things installed. I thought that was worth it. Maybe you don't. Use Chrome if you think addons are too much work.
